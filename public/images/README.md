# Imagens do Projeto

## Hero Section - Imagem de IA

Para usar a imagem personalizada do cérebro digital na HeroSection:

1. Salve a imagem do cérebro digital como `ai-brain.jpg` ou `ai-brain.png` nesta pasta
2. Atualize o caminho em `src/components/HeroSection.jsx`:

```jsx
<img
  src="/images/ai-brain.jpg"
  alt="AI Brain Technology"
  className="w-full h-auto rounded-lg object-cover"
/>
```
