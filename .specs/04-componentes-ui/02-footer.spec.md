# 4.2 Footer

## Descrição
Implementar componente de rodapé do blog.

## Dependências
- Nenhuma

## Etapas

### Etapa 1 — Criar componente Footer
- Criar `src/components/Footer.jsx`
- Layout com:
  - Nome do blog / copyright
  - Ano atual (dinâmico)
  - Links úteis (ex: GitHub, sobre)

### Etapa 2 — Estilização
- Posicionamento fixo na parte inferior (sticky footer)
- Responsivo para mobile e desktop
- Cores e tipografia consistentes com o tema

### Etapa 3 — Integrar no layout
- Adicionar `Footer` no layout principal (abaixo do conteúdo)
- Presente em todas as páginas

## Critérios de Aceite
- [ ] Footer exibe informações do blog e ano atual
- [ ] Footer está posicionado corretamente na parte inferior
- [ ] Footer é responsivo
- [ ] Footer está presente em todas as páginas
